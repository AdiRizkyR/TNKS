# Taman Nasional Kepulauan Seribu (TNKS) - Landing Page

Selamat datang di repositori antarmuka Landing Page pengunjung **Taman Nasional Kepulauan Seribu (TNKS)**. 

Website ini dirancang secara statis untuk memberikan informasi interaktif nan berestetika tentang pesona alam kawasan konservasi pesisir teluk ibukota. Pengunjung dapat memperoleh edukasi terkait status konservasi ekosistem (Terumbu karang, Mangrove, dll), daftar satwa laut rawan, kumpulan regulasi atau aturan resmi hingga tarif wisata PNBP dari Kementerian Lingkungan Hidup dan Kehutanan RI.

## ✨ Fitur Estetika (UI / UX Modifikasi)
- **Struktur Premium Terpisah:** Pemisahan fungsional tata kelola per file (`index.html`, `style.css`, dan `script.js`)
- **Tampilan Kaca Mengapung (Glassmorphism):** Skema visual tembus pandang berbahan buram *frost* untuk menyamankan pembacaan di atas gambar dinamis.
- **Responsive Web Design:** Tampilan konsisten menawan untuk pemakaian di versi *Mobile* menggunakan navbar *hamburger*.
- **Animasi Animate On Scroll (AOS):** Perpindahan antar navigasi ke dalam tata letak yang termuat secara berurutan (*fade up*).
- **Infinite Modals Gallery:** Modifikasi fitur carousel galeri gambar yang dapat diklik tanpa henti secara *loop*.
